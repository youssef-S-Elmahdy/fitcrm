# FitCRM — Simple Client Manager (Frontend Only)

A lightweight, **HTML + CSS** demo CRM for fitness professionals to add clients and browse a sample client list. This project intentionally contains **no JavaScript**: buttons and actions are placeholders.

## Features (Matches Assignment)
- **Page 1: New Client Form**
  - Full Name, Age, Gender, Email, Phone, Fitness Goal, Membership Start Date
  - “Add Client” button (placeholder / no action)
- **Page 2: Client List**
  - Responsive table with **10 sample clients**
  - Columns: Name, Email, Phone, Fitness Goal, Membership Start Date
  - Row actions: **Edit** / **Delete** (placeholders)
  - Search input above the table (placeholder)
- **Responsive Design**
  - Works great on desktop & mobile
  - Uses **Flexbox/Grid** and **media queries**
  - Horizontal scroll for table on small screens
- **Frontend-only**: clean UI, accessible labels, keyboard-friendly

## Tech Stack
- **HTML5**
- **CSS3** (custom, no frameworks)
- Google Fonts (Inter)
